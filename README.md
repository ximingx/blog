个人博客的搭建过程, 大二下半学期

## 项目的启动

首先将项目克隆下来

```bash
$ git clone https://github.com/ximingx/blog.git
```

在 service 文件夹

```bash
$ npm install
$ node app.js
```

在 web 文件夹 

```bash
$ npm install
$ npm run serve
```
